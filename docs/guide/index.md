# 介绍

## 什么是 QVision？

QVision 是一款专为工业应用设计的机器视觉软件，无需编程知识即可实现复杂的视觉检测任务。

## 主要功能

### 缺陷检测
- 表面缺陷检测
- 尺寸测量
- 颜色识别
- 条码/二维码识别

### 质量控制
- 产品分类
- 瑕疵识别
- 一致性检查

### 数据分析
- 实时统计
- 历史数据分析
- 报表生成

## 系统要求

- 操作系统：Windows 10/11
- 内存：4GB 以上
- 摄像头：USB 或网络摄像头
