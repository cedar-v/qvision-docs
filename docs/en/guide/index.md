# Introduction

## What is QVision?

QVision is a machine vision software designed for industrial applications that enables complex visual inspection tasks without programming knowledge.

## Main Features

### Defect Detection
- Surface defect detection
- Dimension measurement
- Color recognition
- Barcode/QR code recognition

### Quality Control
- Product classification
- Defect identification
- Consistency checking

### Data Analysis
- Real-time statistics
- Historical data analysis
- Report generation

## System Requirements

- Operating System: Windows 10/11
- Memory: 4GB or more
- Camera: USB or network camera

For any issues, please contact our technical support team.
