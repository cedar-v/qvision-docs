# Getting Started

## Install Software

1. Download the QVision installer
2. Double-click to run the installation program
3. Follow the prompts to complete installation

## Connect Device

1. Launch QVision software
2. Click "Device Management"
3. Select camera type and connect

## Create Project

1. Click "New Project"
2. Enter project name
3. Select detection type

## Configure Detection

1. Add detection areas in the project
2. Set detection parameters
3. Save configuration

## Start Detection

Click the "Start Detection" button, and the system will automatically perform intelligent analysis.

::: tip Tip
For first-time users, we recommend watching tutorial videos or contacting technical support.
:::
