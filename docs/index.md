# QVision

::: tip 欢迎使用 QVision
无需编程的机器视觉应用软件，让您轻松实现智能检测。
:::

## 产品特点

- **无需编程** - 拖拽式操作，简单易用
- **智能检测** - 支持多种视觉检测算法
- **实时监控** - 实时数据显示和报警
- **灵活配置** - 可自定义检测参数

## 快速开始

1. 下载并安装 QVision 软件
2. 连接您的摄像头设备
3. 创建新的检测项目
4. 配置检测参数
5. 开始智能检测
